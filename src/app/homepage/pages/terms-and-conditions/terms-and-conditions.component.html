2. Game Overview

WOTIRIYE LOTTO is a mobile lottery game where players select numbers between 0 and 9. Winners are determined based on the match between selected numbers and the official draw numbers.

  • Exact Order
  • Any Order
  • Combo Mode (a mix of exact and any order permutations)

4. Choose the way you want to play

    You can play Match 2, Match 3, Match 4, Match 5, or Match 6. Each option simply changes how many digits (0–9) you pick, with unique odds and stake multipliers for each. After choosing which Match to play:

  1. Choose between:
    • Quick Pick: Let our system randomly generate your numbers for you, based on your mobile number.
    • Pick Your Own Numbers: Manually select any numbers from 0–9 for the chosen game type.

      • It’s that simple—decide how many digits you want to match, choose how to pick them, and you’re all set to play!

5. How to Win

At the end of each game round, a winning 3-digit number is drawn. To win, your ticket must match the winning number under one of the following conditions:

  • Exact Order: All digits and positions match exactly.
  • Any Order: All digits match, regardless of position.
  • Combo: Multiple permutations are generated from your selection, and payout is based on applicable match type.
